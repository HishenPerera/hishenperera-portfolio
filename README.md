# 🌐 Hishen Perera – Developer Portfolio

Welcome to my personal portfolio website repository.  
This project showcases my **skills, projects, and experience as a software developer**.  
The portfolio is built using **ASP.NET Core** and deployed using **Railway**.

## 🚀 Live Website
🔗 https://hishenperera.site

## 📌 About the Project
This portfolio website was created to present my development work, technical skills, and projects in a clean and professional interface. It allows visitors, recruiters, and collaborators to explore my work and learn more about my experience.
